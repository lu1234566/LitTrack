# Fundos da Readora Wrapped

Substitua cada arquivo abaixo pela sua arte (PNG 1080x1920, 9:16), **mantendo o mesmo nome**.
Os arquivos atuais são placeholders transparentes (mostram só o gradiente até você trocar).

- 01-intro.jpg        → Intro
- 02-livros.jpg       → Quantidade de livros
- 03-paginas.jpg      → Páginas
- 04-autor.jpg        → Autor do ano
- 05-genero.jpg       → Gênero
- 06-atmosfera.jpg    → Atmosfera / vibe
- 07-mes.jpg          → Mês mais forte
- 08-top5.jpg         → Top 5
- 09-livro-do-ano.jpg → Livro do ano
- 10-maior-livro.jpg  → Maior livro (sinopse)
- 11-final.jpg        → Recap final

O app aplica automaticamente o movimento (Ken Burns + partículas) por cima.
