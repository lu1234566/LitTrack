# Fundos da Readora Wrapped

Substitua cada arquivo abaixo pela sua arte (PNG 1080x1920, 9:16), **mantendo o mesmo nome**.
Os arquivos atuais são placeholders transparentes (mostram só o gradiente até você trocar).

- 01-intro.png        → Intro
- 02-livros.png       → Quantidade de livros
- 03-paginas.png      → Páginas
- 04-autor.png        → Autor do ano
- 05-genero.png       → Gênero
- 06-atmosfera.png    → Atmosfera / vibe
- 07-mes.png          → Mês mais forte
- 08-top5.png         → Top 5
- 09-livro-do-ano.png → Livro do ano
- 10-maior-livro.png  → Maior livro (sinopse)
- 11-final.png        → Recap final

O app aplica automaticamente o movimento (Ken Burns + partículas) por cima.
